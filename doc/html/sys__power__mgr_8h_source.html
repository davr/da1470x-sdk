314 /*
315  * \brief Put the system to idle or sleep or block in a WFI() waiting for the next tick, if neither
316  * idle nor sleep is possible.
317  *
318  * \details Puts the system to idle or sleep, if possible. If an exit-from-idle or a wake-up is
319  * needed, it programs the Timer1 to generate an interrupt after the specified idle or sleep
320  * period. Else, the system stays forever in idle or sleep mode.
321  * If neither idle nor sleep is possible, it blocks in a WFI() call waiting for the next
322  * (already programmed) OS tick to hit.
323  *
324  * \param[in] low_power_periods The number of (prescaled) low power clock periods the OS will be
325  * idle. If it is 0 then the OS indicates that it can block forever waiting for an
326  * external event. If the system goes to sleep, then it can wake up only from an external
327  * event in this case.
328  *
329  * \warning Must be called with interrupts disabled!
330  *
331  */
