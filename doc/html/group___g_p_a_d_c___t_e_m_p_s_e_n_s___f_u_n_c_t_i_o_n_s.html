◆ hw_gpadc_convert_to_celsius_x100()

__STATIC_INLINE int16_t hw_gpadc_convert_to_celsius_x100 ( uint16_t  adc_val)

Convert a 16-bit, left-aligned, raw ADC digital value to temperature. The conversion uses the current GPADC configuration settings. For accurate conversions using this function the ADC should operate in the following configuration:

Positive and negative offset registers = Default (0x200 uncalibrated), SampleTime = 0x02, Oversampling = 64 Samples, Chopping = Enabled and Attenuator = Disabled.

Parameters
[in]adc_valdigital GPADC value
Returns
temperature in hundredths of Celsius degrees (ex. 2540 = 25.4 C)
