Detailed Description

MCIF message buffer.

A message passed to MCIF must be contained in this struct. This struct must be allocated with a size equal to sizeof(struct mcif_message_s) + l, where l is the message length (that in the case of MCIF-ASCII must include the terminating null). The len parameter must contain the length of the message stored in the buffer parameter. When going towards MCIF, the struct will be freed by MCIF. On the other direction, towards the client task, it must be freed by the task.
