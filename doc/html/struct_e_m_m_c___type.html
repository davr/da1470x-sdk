◆ EMMC_ERROR_INT_STAT_R_REG

__IOM uint16_t EMMC_Type::EMMC_ERROR_INT_STAT_R_REG

(@ 0x00000032) This register enables an interrupt when the Error Interrupt Status Enable is enabled and at least one of the statuses is set to 1. Writing to 1 clears the bit and writing to 0 retains the bit unchanged. Signals defined in this register can be enabled by the Error Interrupt Status Enable register, but not by the Error Interrupt Signal Enable register. More than one status can be cleared with a single register write. This register is applicable for an SD/eMMC/UHS-II mode.
