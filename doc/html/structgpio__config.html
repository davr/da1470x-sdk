Detailed Description

GPIO pin configuration.

It's recommended to use HW_GPIO_PINCONFIG and HW_GPIO_PINCONFIG_RESERVE to set pin entries. Each configuration must be terminated using HW_GPIO_PINCONFIG_END macro.

Note
The struct is declared with the compiler attribute __PACKED_STRUCT
