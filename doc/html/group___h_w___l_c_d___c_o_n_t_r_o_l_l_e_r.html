Value:
((!(dg_configDIVN_FREQ % (hz))) ? ((dg_configDIVN_FREQ / (hz))) : \
(!(dg_configRCHS_96M_FREQ % (hz))) ? (((dg_configRCHS_96M_FREQ / (hz))) | HW_LCDC_CLK_RCHS_BIT) : \
(((dg_configPLL160M_FREQ / (hz))) | HW_LCDC_CLK_PLL_BIT))

Macro to calculate the LCDC divider to produce a frequency using the provided source clock.

Due to physical restrictions only the following ranges of frequencies are valid:

Output (interface) clock Source DIVN (32MHz) Source 96MHz Source 160MHz
DPI Parallel 32 MHz - 1 MHz 96 MHz - 3 MHz 160 MHz - 5 MHz
JDI Parallel 16 MHz - 0.5 MHz 48 MHz - 1.5 MHz 80 MHz - 2.5 MHz
Serial 16 MHz - 0.5 MHz 48 MHz - 1.5 MHz 80 MHz - 2.5 MHz

In case of JDI parallel and serial interfaces the divider's value is automatically adapted (divided by 2) to produce the correct frequency

Parameters
[in]hzFrequency in Hz
Note
If the requested frequency is not supported (i.e. there is no LCDC divider to produce the exact frequency) the next available frequency will be selected.
Warning
The application or the adapter (if used) is responsible to turn on the PLL if needed and maintain it as long as needed.
