Detailed Description

Definition of API for the embedded Multi-Media Card (eMMC) Low Level Driver.

Warning
eMMC is not supported in a DA14705 or a DA14706 device variant. A compilation error is produced in case the dg_configUSE_HW_EMMC is accidentally enabled.
